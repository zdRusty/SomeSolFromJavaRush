Условие:
Сериализуй класс Person стандартным способом. При необходимости добавь некоторым полям модификатор transient.

Требования:
- класс Person должен поддерживать интерфейс Serializable
- поле fullName должно быть отмечено модификтором transient
- поле greetingString должно быть отмечено модификтором transient
- поле outputStream должно быть отмечено модификтором transient
- поле logger должно быть отмечено модификтором transient
- поле firstName НЕ должно быть отмечено модификтором transient
- поле lastName НЕ должно быть отмечено модификтором transient
- поле country НЕ должно быть отмечено модификтором transient
- поле sex НЕ должно быть отмечено модификтором transient
