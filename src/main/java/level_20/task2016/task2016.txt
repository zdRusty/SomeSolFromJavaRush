Условие:
Используя минимум изменений кода сделайте так, чтобы сериализация класса С стала возможной.

Требования:
- в классе Solution должен существовать класс А
- в классе Solution должен существовать класс В
- в классе Solution должен существовать класс С
- класс В должен быть потомком класса А
- класс С должен быть потомком класса В
- класс А должен поддерживать интерфейс Serializable
- класс В не должен явно поддерживать интерфейс Serializable
- класс С не должен явно поддерживать интерфейс Serializable
